# TravelGo - Travel Recommendations Website

Final project for IBM JavaScript Programming Essentials. A static travel
recommendations site with Home, About Us, and Contact Us pages, plus beach,
temple, and country recommendations with images.

## Pages
- `index.html` - Home page with beach, temple, and country recommendations
- `about.html` - About Us
- `contact.html` - Contact form with client-side JavaScript handler

## Deploy
Hosted on GitHub Pages.
